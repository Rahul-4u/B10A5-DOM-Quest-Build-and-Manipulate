What is the Document Object Model (DOM)?

The document object model is a programming interface for web documents. It represents the structure of a document as a tree of object. Each mode in the tree corresponds to parts of the document, such as elements, attributes and text

What is event delegation in the context of the DOM, and why is it useful?

Event delegation is a technique in the DOM where you attach a single event listener to a parent element instead of individual listeners to multiple child elements
